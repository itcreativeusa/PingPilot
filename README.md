# PingPilot

AI-powered outreach assistant for generating, saving, and sending personalized LinkedIn and email messages.

## Features

- Generate personalized LinkedIn or email outreach messages
- Choose tone and goal
- Save generated drafts locally in the browser
- Copy messages for LinkedIn or email
- Simple Next.js + TypeScript starter structure

## Setup

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Environment

Create `.env.local`:

```bash
OPENAI_API_KEY=your_openai_api_key_here
```

## Notes

This starter stores drafts in browser local storage. Later you can add Gmail, LinkedIn, database, CRM, authentication, and analytics integrations.
